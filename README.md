# skyrush-legends
SkyRush Legends é um jogo de plataforma 2D com múltiplas fases, personagens únicos e habilidades especiais. O jogador deve atravessar obstáculos, utilizar estratégias e sobreviver até o final enquanto acumula pontos e ativa checkpoints. O jogo inclui sistema de vidas, ranking local e trilha sonora dinâmica gerada no navegador.
