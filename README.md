# 🎮 SkyRush Legends

**SkyRush Legends** é um jogo de plataforma 2D desenvolvido em HTML, CSS e JavaScript puro, focado em habilidade, tempo de reação e estratégia.

---

## 🚀 Sobre o jogo

Corra, salte e sobreviva em um mundo repleto de obstáculos, plataformas móveis e desafios progressivos. Escolha entre diferentes personagens, cada um com habilidades únicas, e avance por múltiplas fases até se tornar um verdadeiro mestre da arena.

---

## 🧠 Mecânicas principais

- Sistema de fases com dificuldade progressiva
- Personagens com habilidades únicas:
  - 🟥 **Runner** — velocidade aumentada
  - 🟣 **Ninja** — salto mais alto
  - ⚫ **Tank** — mais resistência
  - 🟢 **Forest** — velocidade + salto + vidas extras
- Sistema de vidas e dano
- Checkpoints estratégicos
- Obstáculos dinâmicos (serras, rotores, lava, espinhos)
- Inimigos derrotáveis (com recompensa)
- Sistema de pontuação
- Ranking local
- Trilha sonora dinâmica gerada no navegador
